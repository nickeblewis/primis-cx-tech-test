# primis-cx-tech-test

There is no need to run NPM install, as there are no NPM modules included

In order to run the script, make sure you are in the project directory i.e `cd primis-cx-tech-test`

To run the script
`node index.js`

You shall see the results displayed in your terminal, the number of countries imported and the output folder is filled with two new files, one HTML and one CVS. The output folder is generated by the script and is excluded from Git.
